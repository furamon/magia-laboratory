### スゴロクエプレ / グラン・エプレオール / モルゲン氏同人誌

「たかなほがなかよし」「ひめあかがなかよし」だけではすまないエプレ矢印のしっちゃかめっちゃかぶり…！！しょっぱなから私の解像度カセットビジョンかよってなりました（ファミコン以前のゲーム機）\
紅巴ちゃんが愛されていてうれしかったです（１ねん１くみ frmn）\
こんだけみんながみんなを愛していてどろどろにならないのはひとえに作者様の技量と公式様の描写力のたまものだと思います。\
また「同人誌=エロ」という固定観念をぶっ壊されました。こんなほっこり尊いものもあるのですね…コミケも行ったことない同人誌DTなので新体験です。

### 神琳と雨嘉がイノチ感じる話 / 神雨 / 嵩乃朔氏同人誌

まず公式のセリフ持ってくるところからずるいです負けました。勇気が出ない神琳さん、全部お見通しの雨嘉ちゃん、「逸脱したらだめなの？」……百合の教科書～～～！！\
「ラスバレ」ネタだそうですが、いや小説読んだだけの私ですらこんなに尊いんですが。何？変身でも残してるの？？？\
気安く神雨ネタを書きまくっていたのを反省させられました…ドラマの塊だあんたら…報われてくれ…。

### アフターキャッツアンドドッグス / 神雨R18 / ウルツ氏同人誌

「使」ってはいけないえちち本その1．もう何も言いません。指と舌オンリー、ゆりえっちかくあるべし。イノチ、感じるほどにー♪\
作者様が商業百合連載経験者とだけあって すごく えっちで とうとい。「私のような血に穢れた野郎が読んではいけないのでは」というほど。ええい、こんなもにょっとした気持ちになるから百合に深入りしたくなかったのだ！あきらめて沼に沈みます

### Unicorn♡Princess / ひめあかR18 /ことり想氏同人誌

まず読めたことに感謝。作者様が推し作家だったので…！！ひとりゆりえっちバレというのも大好物ですもっとだーグヘッ\
紅巴ちゃんに見守られ、そして互いにひとりでしたのとLove的に好きであることを告白からのゆりえっち。やはり指舌オンリー。そうなんだよ、リリィ同士にマギ交感機なんていらないんだ！\
アサルト成分強めで楽しんでいたため、まずヘルヴォルに惹かれたのでグラン・エプレ自体が個人的にダークホースでした、なのでいろいろとドキドキです。五輪の花々グラン・エプレ、幸せになれ。

### from ME to YOU / ゆずもみ / 錦屡氏同人誌

失礼ながら御台場女学校の知識がなく、登場リリィ自体を存じてなかったので調べるところからスタートでした。二水ちゃんにボコられろ\
楪ちゃんも椛ちゃんもドラマの塊ですね。アサリ裏主人公多すぎだろ！全員主人公が煽り文句じゃないやつかよ！

それはそうと尊い（語彙抹殺）神雨といいたかなほといい、こういう「全部お見通しの相棒兼ふーふ」みたいなのに弱いようです…というかそれは百合好きみんなそうか…。\
「幸せになってほしいが取られたくもない」という感情の吐露もずるいです。ていうかアサルトリリィ全部ずるいよエクゾディア5枚だけのデッキかよ…ちきしょう…！

### 催眠術をかけてみた / 神雨、百合亜×聖恋、船田姉妹 / シャロ氏同人誌

先述通り御台場、さらにルドビコ女学院もリリィの区別もつかない事態なので予習から開始し、そして船田姉妹の設定に先制砲撃を食らいました。重い、重いよリリィ！！聖恋ちゃんも来夢ちゃんに恋しつつも契ったのは百合亜さん…ドラマチック。

百合で催眠…？と最初はよくわかりませんでしたが、読んで納得。ヘテロにばっかり使われる催眠術を「秘めた好意を引き出す」ことで無理なく百合にしています。細かい世界観描写も美しい…！\
神琳さんが自分を好きでうれしい雨嘉ちゃん、全部抱え込む聖恋ちゃんを支えようと尽力する百合亜さん、そして催眠など野暮だった？船田姉妹。六者三様の百合模様。最初警戒した私をCHARMの波動で解体してください。

### アサルトルベルムコンセプトブック / if構想資料集 / 尾花沢軒栄氏公式同人誌？

まず「15年がかりで育て上げた作品の初期構想」というだけでアサリリファン心理を抜いても聖書です。ちょうだい こういうのちょうだい もっと\
こういう「初期構想のリメイクif」ってだけでもう心が躍ります。古のオタクなので。\
というか「ファンタジー戦記が初期着想」ってマジですか…冗談でリリィエムブレムとかアサリリッサーとか言ってたらあたらずといえども遠からずだったとは。やっぱり私がリリィ沼にはまるのはすべて定められた運命だったのか…！！\
「楓さんの元カノが主人公のはずだった」というのも「オウガバトル」の聖騎士ランスロットの歴史かなんかのよう。こういうのもっと見たいです、二水ちゃんふぁいとー！おー！

### アサルトリリィマガジンVol1～3 / アールヴヘイム（壱盤隊）、御台場女学校、私立ルドビコ女学院 / 公式ムック

「電撃新潟奪還戦」も本当に理想で、「アサルト」が「リリィ」の愛を引き立て切っていて…殺伐時代劇しか描けない私にこれらを作った腕を食べさせてください（冗談です）\
ルドビコに至っては設定でやられました。もうあらゆるオタクの急所わかってんなアサルトリリィ！もっとやってくれ！！
