### Windows Defenderの暴走を止める

今やこれで十分なほど高機能なWindows Defenderですが、実は自分を延々ウイルススキャンするという不具合があり、これでメモリリークしたことがあります。\
なので下手順で止めましょう。

1. 「Windowsセキュリティ」アイコンから「ウイルスと脅威の防止」→「ウイルスと脅威の防止の設定」→「除外の追加または削除」と進む
2. 「C:\Program Files\Windows Defender」をフォルダごと除外

参考：[Antimalware Service Executable (MsMpEng)が暴走？抑制・制御方法を解説！](https://webdesign-mania.info/2020/08/antimalware-service-executable.html)

万が一のバックアップ用。システムリカバリまでとなると有料のものしかなく、一番安全そうだったのがこれでした。\
まだ暫定なのでおすすめあればぜひ。

#### **[alt-ime-ahk-mod](https://github.com/nekocodeX/alt-ime-ahk-mod)**

Altキーでかな英切り替え。英語キーボード大好きマンなので必須。

#### **[7-Zip](https://7-zip.opensource.jp/)**

解凍ソフト。まあこんなところを見てくださってる方には説明不要ですな。早い安いうまい。

#### **[FaithType](https://github.com/m13253/FaithType/blob/master/Readme-ja.md)**

システムフォントのアンチエイリアスを直してくれます。疲れない文字にしてくれます。\
更に後述の「Meiryo UIも大っきらい」でフォントサイズを大きくすればモアベター。

#### **[Meiryo UIも大っきらい！！](https://www.tatsu-syo.info/MySoft/WinCust/index.html)**

システムフォント変更ソフト。上述の「FaithType」できれいにしたフォントをこれで大きくしたり、自前フォントを使ったりします。\
Yu Gothic UIかMeiryo UI、あとは後述のIBM Plex Sans JPがオススメ、お好みで。大っきらいじゃないの？は禁句。

#### **[PlemolJP](https://github.com/yuru7/PlemolJP)**

Windows TerminalやVSCode用のプログラミングフォント。\
Nerd Fontを選ぶとターミナルがめちゃくちゃオシャレに。

#### **[IBM Plex Sans JP](https://github.com/IBM/plex/releases)**

いい感じにMacっぽいきれいフォント。先述の「Meiryo大っきらい」によるシステムフォント設定は「Text」がきれいです。ヒンティングはモニター解像度と相談してきれいな方を。

#### **[CLaunch](https://forest.watch.impress.co.jp/library/software/claunch/)**

昔から有名なランチャーアプリ。デスクトップやスタートメニュー、タスクバーをきれいにするかわりにこれで。\
マウスの余ったボタンにホットキー機能応用でキーアサインすると神になれます。

#### **[Clibor](https://chigusa-web.com/clibor/)**

クリップボード拡張ソフト。Windows標準にもあるっちゃありますがこっちのほうが格段に便利。\
定型文機能まであるので使いこなせる人はとことんやってみて自慢してほしいです。

#### **[Upnote](https://getupnote.com/)**

試験運用中のノートアプリ。Notionを使いやすくした感じでなかなかいいです。

#### **[MiniBin](https://www.gigafree.net/utility/recyclebin/minibin.html)**

タスクバー上にゴミ箱を置けます。\

コンパクトリソースモニター。\

## **省電力化（自己責任ゾーン）**

### Bios(UEFI)いじり

IntelかRyzenかでも異なりますが、たいてい電圧制限かエコモードで運用しています。空冷もハイエンドCPUも大好きなわがままマンなので。\
Ryzenは[こちら](https://pc.watch.impress.co.jp/docs/column/shimilabo/1457647.html)が詳しいです。\
メモリOCは「Power Down Enable」的設定をオンにすればよほど無茶でない限り通ります。Memory Context Restore」、メモリトレーニングも常時オンでも初回起動時にはトレーニングしてくれ、起動時間は短縮します。

### [AfterBurner](https://jp.msi.com/Landing/afterburner/graphics-cards)でグラボ電圧いじり

同上。例えばRTX4070ならメモリ+700Mhzのコア+70Mhz、電力80%といった調子。これで使用電力が8割で性能そのままに。\
クロック-200Mhzからのカーブエディターの1000mvを2810Mhzにあわせるでも可（もちろん要調整）で、こちらは試したところ性能1%減ったかな？で消費電力6割に。無論上級者向けなので自己責任で。

#### Sheldon

.zshrc に加えてフォントを落っことして設定しておきましょう。PlemolJPがおすすめ。
下サイト様を参考にしました。

[高速で設定しやすいZsh/BashプラグインマネージャーSheldonの紹介](https://zenn.dev/ganta/articles/e1e0746136ce67)

```shell
# .zshrc
export PATH=$PATH:$HOME/.local/bin
eval "$(sheldon source)"

# History設定
HISTFILE=~/.zsh_history
HISTSIZE=100000
SAVEHIST=100000
setopt share_history
setopt hist_ignore_dups
setopt hist_ignore_all_dups
setopt hist_ignore_space
setopt hist_reduce_blanks

setopt print_eight_bit

# Control+rで履歴検索
function peco-select-history() {
  BUFFER=$(\history -n -r 1 | peco --query "$LBUFFER")
  CURSOR=$#BUFFER
  zle clear-screen
}
zle -N peco-select-history
bindkey '^r' peco-select-history

if [[ -n $(echo ${^fpath}/chpwd_recent_dirs(N)) && -n $(echo ${^fpath}/cdr(N)) ]]; then
    autoload -Uz chpwd_recent_dirs cdr add-zsh-hook
    add-zsh-hook chpwd chpwd_recent_dirs
    zstyle ':completion:*' recent-dirs-insert both
    zstyle ':chpwd:*' recent-dirs-default true
    zstyle ':chpwd:*' recent-dirs-max 1000
fi

# Control+fディレクトリ履歴検索
function peco-cdr () {
    local selected_dir="$(cdr -l | sed 's/^[0-9]\+ \+//' | peco --prompt="cdr >" --query "$LBUFFER")"
    if [ -n "$selected_dir" ]; then
        BUFFER="cd ${selected_dir}"
        zle accept-line
    fi
}
zle -N peco-cdr
bindkey '^f' peco-cdr

# debでDocker履歴検索
alias deb='docker exec -it $(docker ps | peco | cut -d " " -f 1) /bin/bash'
alias dea='docker exec -it $(docker ps | peco | cut -d " " -f 1) /bin/ash'

# Ctrl+]でGHQリポジトリにトベルーラ
function peco-src () {
local selected_dir=$(ghq list -p | peco --query "$LBUFFER")
if [ -n "$selected_dir" ]; then
    BUFFER="cd ${selected_dir}"
    zle accept-line
fi
}
zle -N peco-src
bindkey '^]' peco-src
```
