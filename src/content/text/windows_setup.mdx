## **省電力化**

### Bios(UEFI)いじり

Ryzen 7900X前提です。マザーボードはMSI B650M Mortar。

* CPU Ratio Apply ModeをAdaptive、Ratioを52、PerformanceVoltageを1.175
* メモリ周波数に合わせてSoc VoltageとVDDPを調整（5600MhzならSoc0.975VのVDDP0.925V）
* PBOをAll negative 15
* Scaler 10x
* Max Boost Override +150\~200

こんな感じで温度90℃以下で定格超えパフォーマンス。\
メモリOCは「Power Down Enable」的設定をオンにすればよほど無茶でない限り通ります。「Memory Context Restore」、メモリトレーニングも常時オンでも初回起動時にはトレーニングしてくれ、起動時間は短縮します。

### [Nvidia App](https://www.nvidia.com/ja-jp/software/nvidia-app/)でグラボ電圧いじり

例えばRTX4070なら電圧30%の電力105%（もちろん要調整）で、試したところ消費電力7割かつ性能が定格よりアップ。無論上級者向けなので自己責任で。

#### Fish設定

起動が早いし使いやすいのでFishでいきます。

[https://qiita.com/irongineer/items/e5462d79bcaa903905b7](https://qiita.com/irongineer/items/e5462d79bcaa903905b7)

[https://qiita.com/eduidl/items/dc2f29cac7232a9a97d8](https://qiita.com/eduidl/items/dc2f29cac7232a9a97d8)

```shell
sudo apt install zip unzip -y

sudo apt-add-repository ppa:fish-shell/release-3
sudo apt update
sudo apt install fish -y
chsh -s /usr/bin/fish

curl -sL https://raw.githubusercontent.com/jorgebucaran/fisher/main/functions/fisher.fish | source && fisher install jorgebucaran/fisher
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.12.0
nano ~/.config/fish/config.fish # source ~/.asdf/asdf.fish 追記
source  ~/.config/fish/config.fish
mkdir -p ~/.config/fish/completions; and ln -s ~/.asdf/completions/asdf.fish ~/.config/fish/completions
sudo apt install fzf -y
sudo apt install bat -y
sudo apt install fd-find -y
alias fd='fdfind' && funcsave fd
alias bat='batcat' && funcsave bat
fisher install PatrickF1/fzf.fish
asdf plugin add ghq
asdf install ghq latest
asdf global ghq latest
fisher install decors/fish-ghq
nano ~/.config/fish/config.fish # set GHQ_SELECTOR fzf 追記
fisher install jethrokuan/z
fisher install 0rax/fish-bd
sudo apt install wslu
nano ~/.config/fish/config.fish # export BROWSER=wslview 追記
fisher install edc/bass
```

最後のbassはfishでbashコマンドが打てるやつです。`bass コマンド`で使用可。
